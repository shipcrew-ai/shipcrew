# Task: Create password hashing and JWT utilities

## Description
Create two utility files for password hashing (using Node.js crypto) and JWT token signing/verification (using jsonwebtoken).

## Acceptance Criteria
- [ ] `backend/src/auth/hash.ts` exports `hashPassword(password)` and `verifyPassword(password, storedHash)`
- [ ] hashPassword uses crypto.scrypt with 16-byte random salt, 64-byte key, returns `salt:hash` hex format
- [ ] verifyPassword uses crypto.timingSafeEqual for constant-time comparison
- [ ] `backend/src/auth/jwt.ts` exports `signToken(userId, email)` and `verifyToken(token)`
- [ ] JWT uses `process.env.JWT_SECRET` with fallback `'shipcrew-dev-secret-change-in-production'`
- [ ] JWT expiry is 7 days
- [ ] verifyToken throws on invalid/expired tokens

## Technical Details
- `backend/src/auth/hash.ts` — use `crypto.randomBytes`, `crypto.scrypt`, `crypto.timingSafeEqual`
- `backend/src/auth/jwt.ts` — use `jsonwebtoken` sign/verify
- Payload shape: `{ userId: string, email: string }`
- Install dependency first: `npm install jsonwebtoken @types/jsonwebtoken -w backend`

## Dependencies
- jsonwebtoken npm package (installed as part of this task)
