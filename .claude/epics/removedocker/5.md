# Task: Create JSON field helper for array serialization

## Description
Create a small utility module at `backend/src/lib/json-fields.ts` with two functions that handle serialization between JavaScript arrays and JSON-encoded strings stored in SQLite.

## Acceptance Criteria
- [ ] `parseJsonArray(value: string | string[]): string[]` — returns array as-is if already an array, parses JSON string if string, returns `[]` on parse failure
- [ ] `toJsonString(arr: string[]): string` — returns `JSON.stringify(arr)`
- [ ] Helper handles edge cases: `null`, `undefined`, empty string, malformed JSON all return `[]`
- [ ] Functions are exported and importable by other backend modules

## Technical Details
- New file: `backend/src/lib/json-fields.ts`
- Keep it minimal — two functions, no dependencies
- The `parseJsonArray` function must be defensive since it's the boundary between DB storage and application logic

```typescript
export function parseJsonArray(value: string | string[] | null | undefined): string[] {
  if (Array.isArray(value)) return value;
  if (typeof value !== "string" || !value) return [];
  try {
    const parsed = JSON.parse(value);
    return Array.isArray(parsed) ? parsed : [];
  } catch { return []; }
}

export function toJsonString(arr: string[]): string {
  return JSON.stringify(arr);
}
```
