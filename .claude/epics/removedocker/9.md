## Technical Details

### Quick Start (new):
```markdown
### 1. Clone and install
git clone ... && cd shipcrew && npm install

### 2. Set up environment
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY

### 3. Run the app
npm run dev
# Auto-creates database and seeds on first run

### 4. Open http://localhost:3000
```

### Advanced: PostgreSQL (new section):
```markdown
## Advanced: PostgreSQL (Optional)

For production or if you prefer PostgreSQL:

1. Start PostgreSQL: `docker compose up -d`
2. Update `.env`: `DATABASE_URL=postgresql://devteam:devteam_secret@localhost:5432/devteam`
3. Update `backend/prisma/schema.prisma`: change provider to `"postgresql"`
4. Run: `npm run db:push && npm run db:seed`
```
